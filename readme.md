# API для управления образовательными программами УрФУ

## Описание проекта
Данный сервис предоставляет API для работы с образовательными программами УрФУ и их модулями. Реализованы основные CRUD-операции, а также дополнительные возможности.

## Функциональность
- Управление образовательными программами (создание, редактирование, удаление, просмотр).
- Управление модулями образовательных программ.
- Привязка образовательных программ к институтам.
- Получение списка всех образовательных программ с вложенными модулями.
- Сортировка образовательных программ по `title`, `cypher`, `accreditationDate`, `standard`.
- Базовая авторизация (Basic Auth) для защиты API.
- Обработка ошибок и валидация входных данных.

## Дополнительный функционал
- Добавление модулей к образовательной программе.
- Привязка образовательных программ к институту.
- Реализация сущности администратора (Admin) для аутентификации.

## Технологии
- Java 21
- Spring Boot 3.4.3
- PostgreSQL
- Docker, Docker Compose
- Swagger для документации API
- JUnit для тестирования
